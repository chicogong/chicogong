## ✨ 我的工作

- **🗣️ 语音与 TTS**: 构建流式 ASR → LLM → TTS 管道，韵律微调，支持可中断播放。
- **📡 实时通信**: WebRTC 集成 VAD、打断控制和分块合成，实现低延迟语音交互。
- **产品与智能体**: 以对话为核心的交互设计，结合记忆、工具调用和多模态可视化。
- **工程实践**: 分布式任务调度、配音平台搭建、TTS 生态周边产品。
