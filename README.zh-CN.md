## 🚀 关于我

构建对话式 AI 和实时语音体验，让人们能像与人交流一样自然地与软件对话。比如把 LLM、TTS、实时通信串起来，做技术验证，也能快速产品化。

### 🗣️ 语音与 TTS 工程化

- 构建流式 ASR → LLM → TTS 管道，支持可中断播放，实现低延迟、自然的对话体验
- 拟人化语音合成，精细调整韵律和情感表达，让合成语音听起来真实且富有感染力
- 基于 WebRTC 的交付方案，配备 VAD、打断控制和分块合成，实现流畅的语音交互
- TTS 工程化实践：分布式任务调度、配音平台搭建、TTS 生态周边产品探索
- 提供 TTS 即服务解决方案，探索语音技术的各类应用场景

### 📐 GDS 产品设计迭代

- 以对话为中心的产品设计：记忆、工具调用、守护/兜底路径、以及可视化的行动轨迹。
- 快速迭代流程：场景脚本 → 低保真原型 → 用户访谈/走查 → 上线后用埋点与日志验证。
- 关注度量：对话成功率、TTS 自然度、工具调用可靠性，形成闭环的优化节奏。
